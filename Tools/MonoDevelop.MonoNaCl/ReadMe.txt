After you build MonoNaCl, move the .dll into one of the following directories.

<<< Mac/OSX >>>
~/Library/Application Support/XamarinStudio-4.0/LocalInstall/Addins

<<< Windows Vista/7/8 >>>
~/AppData/Local/XamarinStudio-4.0/LocalInstall/Addins

<<< Linux >>>
~/.local/share/XamarinStudio-4.0/LocalInstall/Addins
