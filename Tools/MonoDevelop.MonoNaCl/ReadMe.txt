After you build MonoNaCl, move the .dll into one of the following directories.

<<< Mac/OSX >>>
~/Library/Application Support/MonoDevelop-3.0/LocalInstall/Addins

<<< Windows Vista/7/8 >>>
~/AppData/Local/MonoDevelop-3.0/LocalInstall/Addins

<<< Linux >>>
~/.local/share/MonoDevelop-3.0/LocalInstall/Addins
