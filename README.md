The ReignSDK is a set of libraries, projects, tools and source code written in C++ and C#, of which are currently defined as follows:

Platforms
{
    Microsoft
    - Windows
    - Metro
    - Xbox360
    - XNA

    Apple
    - OSX
    - iOS

    Google
    - Android
    - NaCl

    Other
    - Linux
}
